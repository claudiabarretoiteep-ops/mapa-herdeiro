Dedicatória. Este livro é para você que já percebeu que o problema não está lá fora. Para quem tentou mudar as circunstâncias e descobriu que elas voltavam. Para quem organizou a agenda, o ambiente, os relacionamentos — e ainda assim o caos interno persistiu. Não é fraqueza reconhecer isso. É o começo da maturidade. Porque governo real nunca começa de fora para dentro. Ele nasce de dentro para fora. E este livro existe para mostrar como esse governo se constrói — com fundamento, com honra e com direção prática. Para você. Para agora.
